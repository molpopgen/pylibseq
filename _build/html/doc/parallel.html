Parallel computationÂ¶

Under the hood, libsequence using Intel’s TBB library for parallelism. By default, TBB will automatically deterimine the optimal number of cores to use. However, that behavior may not be desirable in certain contexts. For example, on managed HPC systems you may be expected to keep your use of cores under control. You make limit the maximum number of threads used as follows:

>>> from libsequence.parallel import scheduler_init
>>> init = scheduler_init(10)

Any value less than one will result in the TBB default number of threads being used.
