libsequence.polytable.freqFilter()¶

Remove all sites with mutation count <= mincount

Parameters:
  • p – An object derived from libsequence.polytable.polyTable
  • mincount – Sites with mutation counts <= this value will be removed. For example, use a value of 2 to remove singletons
  • haveOutgroup – if True, the sequence at position ‘outgroup’ will not be included in determining if a site is monomorphic
  • outgroup – The index of the outgroup sequence in p

Note

If haveOutgroup == True, this is a filter on derived mutation counts, otherwise it is a filter on minor allele counts. If p is of type libsequence.polyTable.simData, this is a filter on derived mutation counts.

libsequence.polytable.isValid()¶

Return True if p only contains the characters A,C,G,T,N,-,0,1.

Parameters:p – An object derived from libsequence.polytable.polyTable

Note

This is not case-sensitive

Example:

>>> import libsequence.polytable as pypt
>>> x = pypt.polySites()
>>> x.assign_sep([0.1,0.2,0.3],["ATC","CGA","AT-"])
>>> pypt.isValid(x)
True

Now, Z is not a valid character:

>>> x.assign_sep([0.1,0.2,0.3],["ATC","CGA","ATZ"])
>>> pypt.isValid(x)
False
class libsequence.polytable.polySites¶

Bases: libsequence.polytable.polyTable

A polymorphism table for Sequence data. A/G/C/T/N/0/1 is the alphabet for analysis.

Note

See libsequence.polytable.polyTable

Constructor

Parameters:
  • x – A list
  • y – A list

If x is None and y is None, and empty object is created.

If y is None, x must be a list of tuples (position, “string”)

if neither x nor y are None, x is a list of positions, and y a list of haplotypes.

Example:

>>> import libsequence.polytable as polyt
>>> x = polyt.polySites()
>>> x = polyt.polySites( [ (0.1,"ATTA"),(0.2,"GGGC") ] )
>>> x = polyt.polySites( [0.2,0.2],["AG","TG","TG","AC"] )

libsequence.summstats.ld()¶

Pairwise “linkage disequilibrium” (LD) statistics

Parameters:
  • p – A libsequence.polytable.polyTable
  • haveOutgroup – if True, then ougtroup is the index of the outroup sequence in p
  • outgroup – The index of the outgroup sequence in p. Not used if haveOutgroup is False
  • mincount – Ignore site pairs where one or both sites have minor alleles occur < mincount times
  • maxDist – Do not consider pairs of sites > maxDist units apart. Default will be the maximum value of a C/C++ double.

Note

This function skips sites with missing data, gaps, etc.

libsequence.summstats.std_nSLiHS()¶

Standardized \(nS_L\) statistic from Ferrer-Admetlla et al. doi:10.1093/molbev/msu077

Parameters:
  • pt – A libsequence.polytable.polyTable
  • minfreq – Ignore markers with frequency < this value
  • binsize – Standardize statistic in frequency bins of this width
  • gmap – A dictionary relating eacy position in pt to its location on a genetic map.
Returns:

A tuple. The first value is max standardized nSL over all bins. The second is max iHS over all bins, where iHS is calculated according to Ferrer-Admetlla et al. The maximums are calculated based on absolute value.
Return type:

float

Note

Only libsequence.polytable.simData types currently supported

class libsequence.extensions.SimDataVec¶

Bases: object

Wrapper for std::vector<Sequence::SimData> (not the Cython extension class libsequence.polytable.simData!!)

This type is useful in the event that you want to process data sets in parallel using OpenMP.

Note

The use case for this type is a Cython-based package extending pylibseq.

Constructor.

Parameters:p – A list of a list of tuples representing the variation data.

Example:

>>> import libsequence.extensions as le
>>> #Create a vector of 5 identical data sets
>>> #with 2 variable sites per data set.
>>> x = [[(0.1,"00100"),(0.2,"11000")]]*5
>>> y = le.SimDataVec(x)
libsequence.msprime.make_simData(g)[source]¶

Construct a libsequence.polytable.simData from the output of msprime.

Parameters:g – The output from msprime

Note

Thanks to Jerome Kelleher for pointing out the quick implementation using msprime >= 0.4.0.

Example:

>>> import msprime as msp
>>> from libsequence.msprime import make_simData
>>> g = msp.simulate(sample_size = 10,Ne=1e6, recombination_rate=1e-8,mutation_rate=1e-8,length=1e4)
>>> s = make_simData(g)
